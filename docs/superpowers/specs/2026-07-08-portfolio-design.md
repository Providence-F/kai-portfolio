## 4. 首页设计（index.html）

### 4.1 视觉

- 背景色：`#0B1A3A`（深蓝，接近 lovegood.cool 首页）
- 顶部：五线谱 + 小猫插画（复用原站 SVG 结构，必要时替换小猫颜色或微调姿态）
- 中部提示文案：
  > 点击上方小猫，听一段关于研究与产品的旋律；长按下方琴键，进入对应篇章。
- 底部：钢琴键盘 SVG
  - 白键文字：`作品集` / `足迹` / `关于我`
  - 可保留英文 `Portfolio` / `Experience` / `About` 作为次要标注

### 4.2 交互

- **点击小猫**：触发一段预置音频（可复用卡农，或替换为更短的旋律）
- **mouseenter 琴键**：琴键下沉 + 播放对应音符
- **mousedown / touchstart**：琴键按下，开始计时
- **长按超过 800ms**：触发 `circle-reveal` 转场，从琴键中心扩散圆形遮罩，跳转至对应页面
- **右上角小猫菜单**：保留按钮，菜单项简化为：
  - 中 / EN（语言切换，如实现）
  - 联系我（滚动或跳转 about.html）

### 4.3 响应式

- 桌面端：完整钢琴键盘，横向展开
- 移动端：保持钢琴隐喻，SVG 缩放或旋转适配窄屏，琴键触控长按跳转

## 8. 设计 Token

沿用 lovegood.cool 的配色与字体系统：

```css
:root {
  /* 背景 */
  --bg-dark: #0B1A3A;
  --bg-light: #F5F1E8;

  /* 文字 */
  --text-primary: #1A1A1A;
  --text-muted: #6A6A6A;
  --text-on-dark: #FAF3E8;

  /* 强调 */
  --accent: #2E5C4F;
  --accent-hover: #1F4036;

  /* 钢琴 */
  --key-white: #FAF3E8;
  --key-black: #0B1A3A;

  /* 字体 */
  --font-serif: 'Playfair Display', serif;
  --font-handwriting: 'Caveat', cursive;
  --font-sans: 'Inter', 'PingFang SC', 'Microsoft YaHei', sans-serif;

  /* 动画 */
  --ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
  --duration-fast: 150ms;
  --duration-normal: 300ms;
  --duration-reveal: 600ms;
}
```

## 9. 交互与动画规格

### 9.1 首页

| 交互 | 触发 | 效果 | 时长 | 缓动 |
|------|------|------|------|------|
| 小猫点击 | click | 播放音频 + 小猫小幅度跳动 | — | — |
| 琴键 hover | mouseenter | 键面向下位移 4px，背景提亮，播放音符 | 150ms | ease-out |
| 琴键按下 | mousedown | 键面持续下沉 | — | — |
| 长按跳转 | 800ms 长按 | 从琴键中心触发 circle-reveal 遮罩 | 600ms | out-expo |
| 页面转场 | 跳转前 | 圆形遮罩扩散至全屏，显示下一页 | 600ms | out-expo |

### 9.2 子页

| 交互 | 触发 | 效果 | 时长 | 缓动 |
|------|------|------|------|------|
| 导航链接 hover | mouseenter | 下划线从左向右展开 | 300ms | out-expo |
| 项目卡片 hover | mouseenter | 图片轻微放大 1.02，阴影加深 | 300ms | out-expo |
| 作品链接 hover | mouseenter | 箭头右移 4px | 200ms | out-expo |
| 页面进入 | load | 内容从下方淡入 24px | 600ms | out-expo |

## 10. 技术实现

### 10.1 文件结构

```
/
├── index.html
├── portfolio.html
├── experience.html
├── about.html
├── css/
│   └── styles.css
├── js/
│   ├── piano.js        # 钢琴交互与音频
│   ├── transitions.js  # 页面转场
│   ├── audio.js        # 音频播放
│   └── data.js         # 项目/履历数据
├── assets/
│   ├── images/         # 项目截图、自画像
│   └── soundfonts/     # 钢琴音色
└── docs/superpowers/specs/2026-07-08-portfolio-design.md
```

### 10.2 技术栈

- 纯静态 HTML/CSS/JS，不引入框架
- SVG 绘制钢琴与五线谱
- Web Audio API 或 SoundFont 播放钢琴音
- CSS clip-path 实现 circle-reveal 转场
- Google Fonts：Playfair Display、Caveat、Inter

### 10.3 响应式断点

- 桌面：>= 1024px
- 平板：768px — 1023px
- 手机：< 768px
